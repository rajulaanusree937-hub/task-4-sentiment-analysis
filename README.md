# task-4-sentiment-analysis



## Overview

This project performs sentiment analysis on Amazon review data using Natural Language Processing (NLP).

## Dataset

Amazon Reviews Dataset

## Tools Used

* Python
* Pandas
* TextBlob
* Matplotlib
* Google Colab

## Features

* Data Loading
* Data Preprocessing
* Sentiment Classification
* Data Visualization
* Insight Generation

## Outcome

A sentiment analysis model was implemented to classify reviews into Positive, Negative, and Neutral categories using NLP techniques.
